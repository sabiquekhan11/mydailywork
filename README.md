# 🚀 Sabique Khan | Full Stack Developer Portfolio

Welcome to my personal portfolio repository! This project showcases my skills, experience, and the premium digital experiences I've built as a Full Stack Developer.

## ✨ Features

- **Modern Glassmorphism UI**: A sleek, futuristic design with vibrant gradients and subtle animations.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Interactive Project Showcase**: A curated list of my best work with detailed descriptions.
- **Downloadable Resume**: A dedicated, printable HTML resume asset linked directly for easy access.
- **Clean Codebase**: Built using modern semantic HTML5 and Vanilla CSS for maximum performance.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS
- **Typography**: Google Fonts (Inter & Outfit)
- **Icons**: Font Awesome 6
- **Animations**: Custom CSS Keyframes
